# Marketing每日资讯功能使用指南

## 🦞 功能简介

Marketing每日资讯是求职准备助手skill的核心功能之一，参考龙虾制作的专业营销资讯整理格式，为求职者提供高质量的营销行业资讯。

### 主要特点

1. **⏰ 每日自动更新**：每天早上8点自动生成最新资讯
2. **🎯 智能筛选**：根据求职目标自动筛选相关资讯
3. **💡 面试导向**：每条资讯附带面试洞察和应用建议
4. **📊 分类整理**：品牌营销、数字营销、整合营销、营销科技、行业趋势
5. **🤝 开放互动**：支持评论区反馈和定制需求
6. **📈 趋势跟踪**：行业动态、竞品分析、技术趋势实时更新

## 🚀 使用方法

### 手动生成资讯

```bash
# 生成Marketing资讯简报
python3 ~/.openclaw/skills/job-preparation-assistant/scripts/marketing_news_collector.py

# 查看今日简报
cat ~/.openclaw/skills/job-preparation-assistant/output/marketing_brief_$(date +%Y-%m-%d).md

# 或查看通用简报
cat ~/.openclaw/skills/job-preparation-assistant/output/daily_brief.md
```

### 自动定时更新

系统已配置定时任务，每天早上8点自动执行：
- 生成Marketing资讯简报
- 保存到本地文件
- 通过如流发送给用户

### 自定义配置

编辑 `config.json` 文件，调整关注领域：

```json
{
  "user_profile": {
    "focus_areas": [
      "品牌营销（品牌策略、品牌传播、品牌建设）",
      "数字营销（社交媒体、内容营销、SEO/SEM）",
      "整合营销（全渠道营销、营销组合）"
    ]
  }
}
```

## 📊 简报结构

每份Marketing资讯简报包含以下部分：

### 1. 今日资讯概览
- 资讯总数
- 更新时间
- 与用户相关的资讯数量

### 2. 精选营销资讯
按分类展示资讯，每条资讯包含：
- 📝 **资讯摘要**
- 🏷️ **相关标签**
- 💡 **面试洞察**：如何在面试中应用
- 📊 **相关性评分**：与求职目标的匹配度
- 📅 **来源和日期**

### 3. 互动与反馈
- 评论区开放
- 用户定制需求
- 反馈渠道

### 4. 明日预告
- 即将更新的内容
- 重点关注领域

### 5. 使用建议
- 面试准备建议
- 案例积累方法
- 趋势把握技巧

## 💡 求职应用建议

### 面试准备

1. **案例积累**
   - 记录优秀的营销案例
   - 分析成功要素
   - 准备自己的见解

2. **趋势把握**
   - 了解行业最新动态
   - 展示专业度和学习能力
   - 准备行业相关问题

3. **技能提升**
   - 关注新技术和新工具
   - 学习最佳实践
   - 实践应用

### 面试洞察应用

每条资讯都附带"面试洞察"建议，例如：

- **"可提及对AI营销趋势的理解"** → 准备AI营销相关案例
- **"展示数据驱动营销的实践经验"** → 准备数据分析案例
- **"讨论情感营销的实际应用"** → 准备品牌营销案例
